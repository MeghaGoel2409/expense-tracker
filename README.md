# Expense Tracker

A full-stack expense tracking application built as a production-style portfolio project.

## Tech Stack

### Backend

- ASP.NET Core Web API
- Clean Architecture
- Entity Framework Core
- ASP.NET Core Identity
- JWT Authentication
- Refresh Token Flow
- Serilog Logging
- Swagger / OpenAPI

### Frontend

- React
- TypeScript
- Vite
- TanStack Query
- Axios
- React Router
- Tailwind CSS
- Zod
- React Hook Form

## Features

- User registration and login
- JWT access token authentication
- Secure refresh token flow
- Expense CRUD
- Category management
- Pagination
- Filtering and sorting
- Dashboard spending summary
- Category breakdown
- Currency-aware expense display
- Global error handling
- Structured logging
